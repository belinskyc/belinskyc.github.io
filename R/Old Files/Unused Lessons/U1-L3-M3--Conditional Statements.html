Conditional Statements

Purpose of lesson

Purpose:

In the previous two modules we looked at different ways to get input values and store these values as a variable [link], and we looked at the six different types of conditional expressions using conditional operators [link]. In this module we will create conditional statements in R that evaluate conditional expressions and output text if the statement is true.  Write a conditional statement in R and execute commands if the statement is true

Concepts in lesson

Concept:

Conditional Statements <BigPic comparison with other types of expressions- put in last module> are essentially decision points (road signs) within a program that determine the direction of the program.  Conditional statements ask a binary question/ with a binary answer yes(true)/no(false) questions like, "Is this fish above 100g?  Is the fish male?  Did the fish get caught in the fall?".  These statements are evaluated by the program and then code is executed based on the answer.  The code can be as simple as outputting text to the screen or as complex as <putting this particular fish into a cluster>.

Examples from lesson

Examples:

In the last module we developed conditional statements for the four following question.

  • Is the fish’s weight greater than 100 grams? (fishWeight > 100)
  • Was the fish caught during the nighttime? (timeCaught == “nighttime”)
  • Did the fish get caught outside the month of April (monthCaught != 4)
  • Was the fish caught in 2012 or before (yearCaught <= 2012)

Now we will put these statements into R code...

The basic structure of an if statement is:

if (#statement to check)
{
# execute the code between the bracket {} if statement is true    
}

  -- next module we will show you what to do if statement is false

Now we will put those four statements into if structures:
- no code is executed in the statement is false... next time

if (fishWeight > 100)  # Is the fish's weight > 100 grams?
{
  output(“It is greater”)  # yes, the weight of the fish is greater than 100 grams
}

if (timeCaught == "nightTime")  # Was the fish caught at night?
{
output("Fish caught at night")  # yes, the fish was caught at night
}

if (monthCaught != 4)  # Was the fish not caught in April?
{
  output("Fish caught in April")  # yes, the fish was not caught in April

}

if (yearCaught <= 2012)  # Was the fish caught on or before 2012?
{
 output("Fish caught before 2012")  # yes, the fish was caught before 2012
}
