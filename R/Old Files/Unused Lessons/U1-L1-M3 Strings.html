Strings and Errors

Purpose:

Create categorical variables that are assigned string values.  Use the mathematical operations on the string and show errors.

Concept:

Learn to do mathematical operations on variables and the many errors that can result from doing them wrong.  Learn about string variables

Examples:

  1. Start new script
  2. Math operation errors:  8+7=a; b=a+7; b = 5+-3; a=2, b=2a
  3. parenthsis
  4. roots and exponentials
  5. Categorical variable: use quote
  6. Explain what happens if you do not use quotes (thinks it is another variable)

Trap: quotes matter in string variables
Extension: How to add two strings together
Shortcut: none

Debugging/Commenting/ Errors:
