Iterative operation: While <moving this to another lesson>

The are times when the number of data points in not given by the input source.  In these cases we use a while() loop.  The while() looks a lot like if() in that while()  also contains a conditional statement:

while ( # some condition is TRUE )
{
    # execute the code in between these brackets
}

Like an if(), the while() will only execute the code in between the curly brackets if the condition is TRUE.  The difference is that while() will keep iterating the execution of the code until the condition is FALSE.  Usually there will be some condition like while ( we are not at the end-of-file ) and when the end-of-file signal is sent, the script exits the while(). 

Trap: The infinite while() loop

In this case, the user will supply the FALSE condition.  Let's make the FALSE condition any negative number. 

while ( # data point value is a non-negative number )
{
    # execute the code in between these brackets
}

So, the while() will iterate as long as the user enters a positive number or zero.  The challenge is when to ask the user for the input

while (dataPoint >= 0)  # will run as long as the user input is 0 or a positive number
{
   numPoints = numPoints + 1               # increase number of points by one
   runningTotal = runningTotal + dataPoint # increase the running total by the data point
}

Extracting information from vectors

When we have vectors representing some experimental variables (in this case, high temperatures), we usually want to do a lot more than just displaying the information. We want to extract information out of the data.  For instance we might to find the maximum of the data or the mean of the data.

Finding the maximum

If we want to find the maximum temperature in our high temp vector, we can use a for() to iterate through each value in the vector, checking each to see if it is the highest.

Essentially the process is:

1) go through each temperature in the vector highTempData using a for()

2) for each index temperature: see if the temperature is higher than the maxTemperature

3) if the temperature is higher, set the maxTemperature to this temperature

Step 1 asks you to iterate through a vector -- so that uses a for().  The code below iterates through each value in the vector highTempData but does not do anything else...

for(day in 1:length(highTempData))
{
 
}

Step 2 asks you to check each vector value and see if its value is greater than maxTemperature

So, inside the for(), we need to put an if() that checks whether the highTempData value for the day is greater than the maxTemperature...

for(day in 1:length(highTempData))
{
  if(highTempData[day] > maxTemperature) 
  {  # start the code associated with the if()
 
  }  # end the code associated with the if()
}

for() and if() both use curly brackets ( { } ) to start and end the code.  In the above code, the if() is nested inside the for().  In the above code, the brackets associated with the if() are colored yellow and indented.  It is good programming practice to indent nested statements so that it is easier to read the code and see where the codeblocks associated with different statements end.

Step 3 asks you to set maxTemperature to the current highTempValue when the highTempValue is greater

for(day in 1:length(highTempData))
{
  if(highTempData[day] > maxTemperature) 
  {
    # set the maximum to the indexed value
    maxTemperature = highTempData[day] 
  } 
}

Now let's develop the whole script to find the maximum temperature....

rm(list=ls())

Finding the maximum temperature from the vector highTempData

We need to "prime the pump", so to speak, buy setting a maximum temperature before we check any value. 

The code:

maxTemperature = 0   # a dummy value

sets a maximum temperature initially to 0 (if this was Alaska, we would probably need to choose a lower number!) and then maxTemperature will be overwritten each time a value from highTempData is greater than the maximum.

The code

  if(highTempData[day] > maxTemperature) 

checks if the current indexed temperature is greater than the current maximum temperature.  If it is then the code

maxTemperature = highTempData[day]

is executed, setting the maximum temperature to the indexed temperature, making this temperature the new maximum temperature
