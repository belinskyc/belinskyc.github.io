RStudio Installation

Purpose:

Download, install, and setup Rstudio Preview. Explain the parts of the panels that will be used in this class.  Show how commenting works and its importance.

Concept:

R is...

Rstudio is...

R is a programming language, RStudio is a wrapper around the programming language.  Similar to the idea that Windows is a wrapper around DOS.  RStudio is designed to emulate the most popular programming environments

Examples:

1) Install R (talk about 32 vs 64 bit) https://cran.r-project.org/

2) Install RStudio https://www.rstudio.com/products/rstudio/download3/

3) Open a script (File -> Open File) that has variables, multiple plots, and good commenting

4) Settings to change (Vibrant Ink)

5) Setting to show:

  • Source
  • Environment
  • Brushes (Plots, Environment)
  • Plots
  • Help
  • Console

Trap: R vs RStudio (where "R" takes place in RStudio)
Extension: 32 vs 64 bit and issue with packages (open source)
